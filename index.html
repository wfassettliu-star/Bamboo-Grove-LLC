Connect to Google Sheets

This tracker stores data in a Google Sheet so you and your husband can share the same records. Complete the one-time setup below.

Step 1. Go to sheets.google.com → create a new blank spreadsheet → name it Bamboo Grove Data
Step 2. Click Extensions → Apps Script
Step 3. Delete everything in the editor, paste in the Code.gs file provided, and click Save (floppy disk icon)
Step 4. Click Deploy → New Deployment → Web App
   • Execute as: Me
   • Who has access: Anyone
   • Click Deploy → authorize when prompted → copy the URL
Step 5. Paste the URL below and click Connect.
Your husband pastes the same URL on his device — you'll both see the same data.
Using iPhone or iPad? Safari blocks network requests from locally opened files. Open the app from your GitHub Pages URL instead of the downloaded file — go to your GitHub repository, enable Pages (Settings → Pages → Branch: main), and use the https://username.github.io/repo-name/ URL on any browser, any device.
